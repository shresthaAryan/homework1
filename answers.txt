What is the difference between a merge and a rebase?
Merge and rebase are very similar in a way as they integrate changes from one branch into another branch. Rebase moves a feature brain into a master rewriting the history whereas Merge adds a new commit, preserving the history.
=======
What is a remote repository?
A remote repository is a git repository that is hosted on the internet. Unlike local repository, it does not have a working directory and can be accessed by multiple people.
